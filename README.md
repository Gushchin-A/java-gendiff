### 📑 🔍 Gendiff
Diff Tool – a program that determines the difference between two data structures.

---
### Project launch:
```bash
make build     # project assembly
make run       # run via Gradle
make install   # create distribution
make run-dist  # run installed CLI version

# or run directly if symbolic link exists
./gendiff # path to file
```
